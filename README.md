# Memodian

Memodian은 Markdown으로 빠르게 메모를 작성하고 관리하는 iOS/iPadOS용 메모 앱입니다.

## 주요 기능

- Markdown 기반 메모 작성
- 메모 목록 자동 정렬
- Markdown 미리보기
- 코드 블록 하이라이트
- 위키 링크 형식 지원
- 기기 저장소 및 iCloud 문서 저장소 사용

## 사용 방법

1. 앱을 실행합니다.
2. 우측 상단의 `+` 버튼을 눌러 새 메모를 만듭니다.
3. Markdown 형식으로 내용을 작성합니다.
4. `Preview` 탭에서 렌더링된 문서를 확인합니다.
5. 뒤로 가기 버튼 또는 스와이프 동작으로 목록으로 돌아가면 내용이 저장됩니다.

## 데이터 저장

Memodian은 사용자의 메모를 앱의 문서 저장소에 Markdown 파일로 저장합니다. iCloud 문서 저장소를 사용할 수 있는 환경에서는 사용자의 iCloud 계정을 통해 메모가 동기화될 수 있습니다.
